# AlgoTasker Documentation

## Project Overview

AlgoTasker is a smart task scheduling system that automatically schedules your to-dos based on urgency, estimated time, and due date using scheduling algorithms.

## Features

- Task management (CRUD operations)
- Three scheduling algorithms:
  - Shortest Job First (SJF)
  - Round Robin
  - Priority Queue
- Algorithm visualization
- User authentication
- PWA support with offline functionality

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   cd server && npm install
   cd ../client && npm install
   ```
